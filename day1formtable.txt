## ğŸ§¾ **Part 3: HTML Forms â€“ Detailed Explanation**

### âœ… What is a Form?

* An HTML **form** is used to **collect user input**.
* Input can be: text, selections, checkboxes, files, etc.
* It sends data to a server (or can be processed with JavaScript).

---

### ğŸ§± Basic Structure

```html
<form action="/submit" method="POST">
  <!-- Form inputs go here -->
</form>
```

| Attribute | Purpose                                |
| --------- | -------------------------------------- |
| `action`  | URL where form data is sent            |
| `method`  | HTTP method: `GET` (default) or `POST` |

---

### ğŸ§© Common Form Elements

| Element                              | Usage                          |
| ------------------------------------ | ------------------------------ |
| `<input type="text">`                | Single line text input         |
| `<input type="email">`               | Email input (validates format) |
| `<input type="password">`            | Password field                 |
| `<input type="checkbox">`            | Checkbox                       |
| `<input type="radio">`               | Radio buttons                  |
| `<input type="file">`                | File upload                    |
| `<textarea>`                         | Multiline input                |
| `<select><option>`                   | Dropdown menu                  |
| `<button>` / `<input type="submit">` | Submit button                  |

---

### âœï¸ **Example: Contact Form**

```html
<form action="/submit" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required><br><br>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email"><br><br>

  <label for="message">Message:</label><br>
  <textarea id="message" name="message" rows="4"></textarea><br><br>

  <label>Gender:</label>
  <input type="radio" id="male" name="gender" value="male">
  <label for="male">Male</label>
  <input type="radio" id="female" name="gender" value="female">
  <label for="female">Female</label><br><br>

  <label for="skills">Select Skills:</label><br>
  <input type="checkbox" id="html" name="skills" value="HTML"> HTML
  <input type="checkbox" id="css" name="skills" value="CSS"> CSS<br><br>

  <label for="country">Country:</label>
  <select name="country" id="country">
    <option value="india">India</option>
    <option value="usa">USA</option>
  </select><br><br>

  <button type="submit">Submit</button>
</form>
```

---

### ğŸ¯ Good Practices

* Use `label` elements with `for` attributes.
* Use `name` attributes â€“ required for submitting form data.
* Use `required`, `min`, `max`, `pattern` for validations.
* Avoid putting form fields outside the `<form>` tag.

## ğŸ—‚ï¸ **Part 4: HTML Tables â€“ Detailed Explanation**

### âœ… What is a Table?

* A **table** organizes data in **rows** and **columns**.
* Defined using the `<table>` tag and its children.

---

### ğŸ§± Table Structure

| Tag       | Purpose                   |
| --------- | ------------------------- |
| `<table>` | Main container            |
| `<tr>`    | Table row                 |
| `<th>`    | Table header cell         |
| `<td>`    | Table data cell           |
| `<thead>` | Header section (optional) |
| `<tbody>` | Body section              |
| `<tfoot>` | Footer section            |

---

### âœï¸ **Example: Student Marks Table**

```html
<table border="1">
  <thead>
    <tr>
      <th>Name</th>
      <th>Subject</th>
      <th>Marks</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Ravi</td>
      <td>Math</td>
      <td>95</td>
    </tr>
    <tr>
      <td>Sneha</td>
      <td>Science</td>
      <td>89</td>
    </tr>
  </tbody>
</table>
```

---

### ğŸ§© Additional Features

* `colspan` and `rowspan` to merge cells:

* Styling with CSS:

